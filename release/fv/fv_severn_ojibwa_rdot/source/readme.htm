ᐊᓂᔑᓂᓂᒧᐏᐣ (Severn Ojibwa, right w-dot) Keyboard

This keyboard is designed for the ᐊᓂᔑᓇᐯᒧᐎᓐ (Ojibwa) language of the Eastern Subarctic region of Canada. This is part of a set of keyboards from FirstVoices.

This keyboard uses a-finals and the right w-dot. For Western-finals and left w-dot, see the ᐊᓂᔑᓂᓂᒧᐎᐣ (Severn Ojibwa, left w-dot) keyboard.

For i-finals, see the NAME (left w-dot) keyboard or the NAME (right w-dot) keyboard.

For a-finals, see the ᐊᓂᔑᓇᐯᒧᐎᓐ (Ojibwa, left w-dot) keyboard or the ᐊᓂᔑᓇᐯᒧᐏᓐ (Ojibwa, right w-dot) keyboard.
