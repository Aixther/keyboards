Háiɫzaqvḷa Keyboard

This keyboard is designed for the Háiɫzaqvḷa language of the BC Coast region of Canada. This is part of a set of keyboards from FirstVoices.

NOTE: This version of the keyboard is designed to work with Noto Heiltsuk, and contains unreleased unicode characters.

This keyboard will NOT work as intended on computers that do not have Noto Heiltsuk installed and is not advised for use on social media, texting, or emailing.

The Noto Heiltsuk font will be installed automatically alongside the keyboard - no further action needs to be taken.
