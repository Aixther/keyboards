This is a bilingual English/SENĆOŦEN keyboard. To type in SENĆOŦEN on Mac or PC, press CapsLock and type with uppercase letters.

  • To type the acute accent use the Apostrophe key ’ — Ć is typed C + Apostrophe ’
  • To type the slash accent use the Slash key / — Ⱥ is typed A + Slash /
  • To type the dash accent use the Hyphen key - — Ŧ is typed T + Hyphen -
  • To type the low line accent use the Semicolon key ; — Ṉ is typed N + Semicolon ;
  • The original English punctuation keys can be typed by holding down the Ctrl+Alt or Option key. The slash / is typed Ctrl+Alt+Slash (Windows) Option+Slash (Mac).
