• The special characters of SGUUXS have been assigned to punctuation keys.
  • All accents are typed after the base character: a̱ is typed a then Hyphen -
  • To type the low line use the Hyphen Key — ḵ is typed k then Hyphen -
  • To type the diaeresis accent use the Left-Bracket Key [ - ü is typed u then Left-Bracket [
  • üü can be typed uu then Left-Bracket [
  • All changed punctuation can type their original value by holding down the Right-Alt or Option key. The Left-Bracket [ is typed Right-Alt+Left-Bracket (Windows) Option+Left-Bracket (Mac).

Development of this keyboard layout was funded by FirstVoices and the First Peoples’ Heritage Language and Culture Council. This keyboard is provided free of charge.
