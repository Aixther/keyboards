This keyboard is designed for the Sḵwx̱wú7mesh sníchim language of the BC Coast region of Canada.

If square boxes are displayed instead of characters when using this keyboard (and in the keyboard layouts below), please read our troubleshooting guide.

Desktop Keyboard Layout

  • All accents are typed after the base character: w̓ is typed w then Backquote `
    • To type the acute accent use the Semicolon key ; — á is typed a then Semicolon ;
    • To type the combining glottalized accent use the Backquote key ` (next to the 1 key) — m̓ is typed m then Backquote `
      The weak glottal is typed by using the Apostrophe key '
    • To type the underline accent (below macron) use the Hyphen key — ḵ is typed k then Hyphen - The letters ḵ, x̱ also have their own keys: left bracket [, right bracket ]
  • The letter ḵ can be typed with the Left Bracket [ key.
  • The letter x̱ can be typed with the Right Bracket ] key.
  • The weak glottal ’ is typed with the Apostrophe key '.
  • All changed punctuation can type their original value by holding down the Right-Alt or Option key.
    The Semicolon ; is typed RightAlt+Semicolon (Windows) Option+Semicolon (Mac).
