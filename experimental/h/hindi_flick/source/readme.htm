Hindi Flick

A mobile keyboard layout designed for Hindi, utilizing a flick input method.

Barakunji is a mobile keyboard layout based upon the Devanagari varnamala's (alphabet) clustering of syllables. It is intended to be usable by anyone who knows the varnamala. It is inspired by the flick-based layouts popular in Japan and (to a lesser extent) Korea, as well as Vietnamese TELEX with regards to diacritic rules and magic.

There are 16 total keys, with 12 in the center being swipeable in multiple directions, longpress-able and multitap-able.
