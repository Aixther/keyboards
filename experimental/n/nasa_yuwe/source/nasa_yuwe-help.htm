Guía para el uso del teclado Nasa Yuwe de Keyman

Teclas inactivas

En el teclado Nasa Yuwe hay tres teclas inactivas:

]

para aplicar una tilde sobre los vocales para indicar un vocal nasal
'

para aplicar un acento sobre los vocales
"

para producir c-cedilla, glotales y comillas

para aplicar una diéresis sobre la letra u

para producir comillas

Si desea producir un punto y coma(;) o un colon (:), después de apretar la tecla para producir la ñ o Ñ, simplemente repita la misma tecla una segunda vez o use la barra espaciadora. (Esta última producirá la puntuación seguida por un espacio.) Esto funcionará bien ya que las letras ñ y Ñ nunca ocurren dos veces consecutivamente ni al final de una palabra.
